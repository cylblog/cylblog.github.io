Css特性

发表于 2021-08-21 更新于 2021-09-19 分类于 前端 阅读次数：

  • 这周学习的是css的四大特性，接下来是学习的内容。



    css主要有一下四大特性：

    • 层叠性
    • 继承性
    • 优先级
    • 特殊性

​

  • 首先介绍的是层叠性

    所谓层叠性是指样式的叠加。例如：如果一个属性通过两个相同选择器设置到同一个元素上，那么这个时候一个属性就会被另一个属性层叠掉。

一般情况下，如果出现样式冲突，最会按照以下原则执行

  1. 遵循的原则是就近原则。哪一个样式离着结构近，就执行哪一个样式
  2. 样式不冲突，不会层叠

口诀：长江后浪推前浪，前浪死在沙滩上。

​

​

  • 其次是继承性

所谓继承性就是书写css样式时，子标签会继承父标签的某些样式。如果想要设置一个可继承的属性，只需要将它应用于父元素即可。

使用继承的好处：

  • 简化代码，降低css的复杂性，子元素可以继承父元素的样式
  • 这里的继承是指属性的继承，不要和权重的继承搞混淆，权重的继承为0不代表就不继承。

​

​

  • 然后是css 的优先级

定义css样式时，经常出现两个或者更多的规则应用在一个元素上，这时会出现优先级的问题。

注意事项：

  • 继承样式的权重为0。即在嵌套结构中，不管父元素的样式权重多大，被子元素继承时，他的权重都为0，也就是说子元素定义的样式会覆盖继承来的样式

  • 行内样式优先。应用style属性的元素，其行内元素的权重非常高，可以理解为大于100。总之，他拥有比其他的选择器更高的优先级。

  • 权重相同的时候，css遵循就近原则。也就是说靠近元素样式具有非常大的优先级，或者说排在最后的优先级最大。

  • css定义了一个叫做 !important的命令，被赋予最大的优先级。也就是说无论权重如何以及样式位置的远近，!important都具有最大的优先级。但是!important应用时应当慎重，不到万不得已尽量少用。

关于权重，我们需要一套计算公式去计算，这个就是css特性。它是一个权衡css值优先级的一个标准。

特殊性是用一个四位数来表示的，就好像四个级别，值从左到右，左面的值最大，依次往左边递减。
