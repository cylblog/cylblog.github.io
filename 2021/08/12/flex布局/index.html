flex布局

Posted on 2021-08-12 Edited on 2021-08-13

这一周学习的是flex布局，以下是学习总结：

  • flex布局表示弹性布局，为盒状模型提供最大的灵活性。

    适用范围：任何一种元素都可以指定为flex布局。

    注意点：1. 如果是webkit内核的浏览器，需要加上-webkit前缀

    1. 在父元素设置为flex布局之后，子元素的float、clear、vertical-align属性都将失效，

      所以在flex布局时不应该先设置子元素布局后在使用。

  • 基本概念

    容器：采用flex布局的元素被称为容器

    项目：在flex布局中的子元素被称为项目

  • 设置在容器上的常用六大属性：

    • flex-direction
    • flex-wrap
    • flex-flow
    • justify-content
    • align-items
    • align-content

1.flex-direction

设置容器主轴的方向，包含四个属性

  1. row：默认值，表示沿水平方向，由左到右

  2. row-reverse：表示沿水平方向，由右到左

  3. column：表示垂直方向，由上到下

  4. column：表示垂直方向，由下到上

2.flex-wrap

设置当前项目中一行无法显示的时候如何处理，包含三个属性

  1. wrap：正常换行，第一个位于第一行的第一个

  2. nowrap：表示不换行

  3. wrap-reverse：向上换行，第一个位于下方

3.flex-row

该属性是flex-direction和flex-wrap属性的缩写，默认为【row，nowrap】

4.justify-content

用于项目在容器中的对齐方式，包含五个属性

  1. flex-start：默认值，左对齐

  2. flex-end：右对齐

  3. center：居中对齐

  4. space-between：两端对齐

  5. space-around：两个项目的两侧间距相等

5.align-items

设置项目在交叉轴上是如何对齐显示的，包含五个属性

  1. flex-start：交叉轴的起点对齐

  2. flex-end：交叉轴的重点对齐

  3. center居中对齐

  4. baseline：项目的第一行文字的基线对齐

  5. stretch：如果项目未设置高度或者高度为auto，将占满整个高度

  • 设置在项目上的一些属性：
  • order：设置项目排序的位置，默认为0数值越小越靠前
  • flex-grow：控制当前项目是否被放大。默认为0，即使有剩余空间也不放大；设为1，平均分摊后放大

  • flex-shrink：表示元素的缩小比例。默认为1，如果空间不够用时，所有项目同比缩小。若设为0，空间

    不如时该项目也不缩小。

  • flex-basis：表示项目占据主轴空间的值。默认为auto，表示当前默认项目的大小。如果设为一个固定值，则该项目在容器中占据固定的大小。
  • flex：flex-grow、flex-shrink、flex-basis的简写，默认为： 0 1 auto
  • align-self：表示当前项目可以和其他项目拥有不一样的对齐方式
