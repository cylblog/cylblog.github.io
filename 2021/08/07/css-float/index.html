浅析 CSS float布局

发表于 2021-08-07 更新于 2021-09-19 分类于 前端 阅读次数：

学习css的时候看到float元素的时候很懵逼，于是上网学习了一波，现在简单分享一下自己的学习成果。

  • 首先要知道，在html中div是块级元素，在页面中独占一行，自上而下排。如图：

我们从图中可以看出，即使每个div后面有很多的空间，后面的元素也不会上来填充，这就是标准流的的div。

而float可以理解为让某个div元素脱离标准流，漂浮在标准流之上，和标准流不是一个层次。

  • 比如说，上图中的div2设置为float，但是div1、div3、div4仍然在标准流当中，所以div3会自动上移，占据div2的位置，如图：

图中的div3相比之前变“矮”了，这是因为div3上移了。不过要注意的是我们这里设置的是左浮动（float:left），为了看的更清楚些，我们设置把div2设置成右浮动：

此时的div2漂浮在右边，div3不会被div2遮挡。

  • 在上面我们只设置了一个float，如果我们设置多个float会是什么样子呢？

如图，我们把div2和div3都设置为左浮动：

我们可以看到div3紧跟在div2后面，而div4上移了。

这是由于div2和div3浮动，它们不再属于标准流，因此div4会自动上移，与div1组成一个新的标准流，

而浮动是漂浮在标准流之上，因此div2遮挡了div4.

  • 重点来了，当同时对div2、div3设置了浮动之后，div3紧跟div2，div2在每个例子中都是浮动的，但是都没有j紧跟在div1的后面。这里我们可以得出一个重要的结论：假设某个div元素A是浮动的，如果A的上一个元素也是浮动的，那么A元素会紧跟在上一个元素的后边（如果一行放不下这两个元素，那么A元素会被挤到下一行）。如果A元素上一个元素是标准流中的元素，那么A的相对垂直位置不会改变，也就是说A的顶部总是和上一个元素的底部对齐。

为了更好理解，

  1. 现在我们把div2、div3、div4都设置成左浮动：

我们看图再来分析一波：先从div4开始，div4发现上边的div3是浮动的所以紧跟div3；div3发现上边的div2也是浮动，所以紧跟div2；div2发现上边的div1是标准流中的元素，因此div2的相对垂直位置不变，顶部元素仍然和底部对齐。

  1. 如果设置成右浮动：

这里我们只需要注意靠近页面的边缘是前就可以。

  1. 设置div2和div4为左浮动：

1）由于div1是标准流，div2不动，由于div2是浮动的，所以div3上移了。

2）div4也是浮动的，div3是标准流，div3的底部和div4的顶部始终对齐，所以div4会跟着div3上移。

  • 最后，还有一个清除浮动的功能，清除浮动可以理解为打破横向排列

假设页面中只有div1和div2且都是左浮动，如图：
